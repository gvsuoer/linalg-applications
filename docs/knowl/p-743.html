A word of caution: the knight can only make complete moves, so we are restricted to integer (either positive or negative) values for \(x_1\text{,}\) \(x_2\text{,}\) \(x_3\text{,}\) and \(x_4\text{.}\) You can use the GeoGebra app at geogebra.org/m/dfwtskrj to see the effects the weights have on the knight moves. We should note here that since addition of vectors is commutative, the order in which we apply our moves does not matter. However, we may need to be careful with the order so that our knight does not leave the chess board.
