Let \(C[a,b]\) be the set of all continuous real valued functions on the interval \([a,b]\text{.}\) If \(f\) is in \(C[a,b]\text{,}\) we can extend \(f\) to a continuous function from \(\R\) to \(\R\) by letting \(F\) be the function defined by

\begin{equation*} F(x) = \begin{cases}f(a) \amp \text{ if } x \lt a \\ f(x) \amp \text{ if } a \leq x \leq b \\ f(b) \amp \text{ if } b \lt x \end{cases}\text{.} \end{equation*}

In this way we can view \(C[a,b]\) as a subset of \(\F\text{,}\) the vector space of all functions from \(\R\) to \(\R\text{.}\) Verify that \(C[a,b]\) is a vector space.
