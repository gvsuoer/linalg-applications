Task 22.8.a.i.

\(\det(A^{-1})\)

Solution.

Assume that \(\det(A)=3\) and \(\det(B)=2\text{.}\)

Since \(\det(A) \neq 0\text{,}\) we know that \(A\) is invertible. Since \(1 = \det(I_n) = \det(AA^{-1}) = \det(A) \det(A^{-1})\text{,}\) it follows that \(\det(A^{-1}) = \frac{1}{\det(A)} = \frac{1}{3}\text{.}\)
