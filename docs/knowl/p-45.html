We can use any labels for the variables in a linear equation that we like, e.g., \(I_1\text{,}\) \(x_1\text{,}\) \(t_1\text{,}\) and you should become comfortable working with variables in any form. We will usually use subscripts, as in \(x_1, x_2, x_3, \ldots\text{,}\) to represent the variables as this notation allows us to have any number of variables. Other examples of linear equations are

\begin{equation*} x+2y=4 \qquad \text{ and } \qquad \sqrt{2}x_1-3x_2=\frac{1}{4}x_3+\pi \,\text{.} \end{equation*}
