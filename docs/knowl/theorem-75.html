Theorem 31.3.

Let \(V\) be any vector space with identity \(\vzero\text{.}\)

  • \(0 \vv = \vzero\) for any vector \(\vv\) in \(V\text{.}\)

  • The vector \(\vzero\) is unique.

  • \(c \vzero = \vzero\) for any scalar \(c\text{.}\)

  • For any \(\vv\) in \(V\text{,}\) the additive inverse of \(\vv\) is unique.

  • The additive inverse of a vector \(\vv\) in \(V\) is the vector \((-1)\vv\text{.}\)

  • If \(\vu\text{,}\) \(\vv\text{,}\) and \(\vw\) are in \(V\) and \(\vu + \vw = \vv + \vw\text{,}\) then \(\vu = \vv\text{.}\)
