The number of rows and columns of a matrix is called the size of the matrix, so an \(m \times n\) matrix has \(m\) rows and \(n\) columns. If we label the entry in the \(i\)th row and \(j\)th column of a matrix \(A\) as \(a_{ij}\text{,}\) then we write \(A = [a_{ij}]\text{.}\)
