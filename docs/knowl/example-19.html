Example 10.3.

For each of the following matrices \(A\text{,}\)

  • Use appropriate technology to find the reduced row echelon form of \([A \ | \ I_3]\text{.}\)

  • Based on the result of part (a), is \(A\) invertible? If yes, what is \(A^{-1}\text{?}\) If no, explain why.

  • Let \(\vx = \left[ \begin{array}{c} x_1\\x_2\\x_3 \end{array} \right]\) and \(\vb = \left[ \begin{array}{c} 5\\4\\1 \end{array} \right]\text{.}\) If \(A\) is invertible, solve the matrix equation \(A \vx = \vb\) using the inverse of \(A\text{.}\) If \(A\) is not invertible, find all solutions, if any, to the equation \(A \vx = \vb\) using whatever method you choose.

(a)

\(A = \left[ \begin{array}{crr} 1\amp 2\amp 3 \\ 1\amp -1\amp -1 \\ 1\amp 0\amp 1 \end{array} \right]\)

Solution.

With \(A = \left[ \begin{array}{crr} 1\amp 2\amp 3 \\ 1\amp -1\amp -1 \\ 1\amp 0\amp 1 \end{array} \right]\text{,}\) we have the following.

  • The reduced row echelon form of \([A \ | \ I_3]\) is

    \begin{equation*} \left[ \begin{array}{ccc|rrr} 1\amp 0\amp 0\amp \frac{1}{2}\amp 1\amp -\frac{1}{2} \\ 0\amp 1\amp 0\amp 1\amp 1\amp -2 \\ 0\amp 0\amp 1\amp -\frac{1}{2}\amp -1\amp \frac{3}{2} \end{array} \right]\text{.} \end{equation*}

  • Since \(A\) is row equivalent to \(I_3\text{,}\) we conclude that \(A\) is invertible. The reduced row echelon form of \([A \ | \ I_3]\) tells us that

    \begin{equation*} A^{-1} = \frac{1}{2} \left[ \begin{array}{rrr} 1\amp 2\amp -1 \\ 2\amp 2\amp -4 \\ -1\amp -2\amp 3 \end{array} \right]\text{.} \end{equation*}

  • The solution to \(A \vx = \vb\) is given by

    \begin{equation*} vx = A^{-1} \vb = \frac{1}{2} \left[ \begin{array}{rrr} 1\amp 2\amp -1 \\ 2\amp 2\amp -4 \\ -1\amp -2\amp 3 \end{array} \right] \left[ \begin{array}{c} 5\\4\\1 \end{array} \right] = \left[ \begin{array}{r} 6\\7\\-5 \end{array} \right]\text{.} \end{equation*}

(b)

\(A = \left[ \begin{array}{crr} 1\amp 2\amp 5 \\ 1\amp -1\amp -1 \\ 1\amp 0\amp 1 \end{array} \right]\)

Solution.

With \(A = \left[ \begin{array}{crr} 1\amp 2\amp 5 \\ 1\amp -1\amp -1 \\ 1\amp 0\amp 1 \end{array} \right]\text{,}\) we have the following.

  • The reduced row echelon form of \([A \ | \ I_3]\) is

    \begin{equation*} \left[ \begin{array}{ccc|crr} 1\amp 0\amp 1\amp 0\amp 0\amp 1 \\ 0\amp 1\amp 2\amp 0\amp -1\amp 1 \\ 0\amp 0\amp 0\amp 1\amp 2\amp -3 \end{array} \right]\text{.} \end{equation*}

  • Since \(A\) is not row equivalent to \(I_3\text{,}\) we conclude that \(A\) is not invertible.

  • The reduced row echelon form of \([A \ | \ \vb]\) is

    \begin{equation*} \left[ \begin{array}{ccc|c} 1\amp 0\amp 1\amp 0 \\ 0\amp 1\amp 2\amp 0 \\ 0\amp 0\amp 0\amp 1 \end{array} \right]\text{.} \end{equation*}

    The fact that the augmented column is a pivot column means that the equation \(A \vx = \vb\) has no solutions.
