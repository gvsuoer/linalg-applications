The result of ActivityÂ 24.5 is that if the columns of a square matrix \(P\) form an orthonormal set, then \(P^{-1} = P^{\tr}\text{.}\) This makes calculating \(P^{-1}\) very easy. Note, however, that this only works if the columns of \(P\) form an orthonormal basis for \(\Col P\text{.}\) You should also note that if \(P\) is an \(n \times n\) matrix satisfying \(P^{\tr}P = I_n\text{,}\) then the columns of \(P\) must form an orthonormal set. Matrices like this appear quite often and are given a special name.
