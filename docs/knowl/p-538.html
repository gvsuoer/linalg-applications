Let \(x_1\) be the number of molecules of \(\text{C} _2\text{H} _6\text{,}\) \(x_2\) the number of molecules of \(\text{O} _2\text{,}\) \(x_3\) the number of molecules of \(\text{ CO } _2\text{,}\) and \(x_4\) the number of molecules of \(\text{H} _2\text{O}\) in the reaction. We can then represent this reaction as

\begin{equation*} x_1 \text{C} _2\text{H} _6 + x_2 \text{O} _2 \to x_3 \text{ CO } _2 + x_4 \text{H} _2\text{O}\text{.} \end{equation*}
