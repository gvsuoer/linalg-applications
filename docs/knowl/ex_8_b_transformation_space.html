Exercise 6.

Let \(V\) and \(W\) be vector spaces and define \(L(V,W)\) to be the set of all linear transformations from \(V\) to \(W\) as in Exercise 14 of Chapter 37. The set \(L(V,W)\) is a vector space with the operations as as follows for \(S\) and \(T\) are in \(L(V,W)\) and \(c\) a scalar:

\begin{equation*} (S+T)(\vv) = S(\vv) + T(\vv) \ \ \text{ and } \ \ (cT)(\vv) = cT(\vv) \end{equation*}

for all \(\vv\) in \(V\text{.}\) If \(\dim(V) = n\) and \(\dim(W) = m\text{,}\) find the dimension of \(L(V,W)\text{.}\) (Hint: Let \(\CB\) be a basis for \(V\) and \(\CC\) a basis for \(W\text{.}\) What can be said about the mapping that sends \(T\) in \(L(V,W)\) to \([T]_{\CB}^{\CC}\text{?}\) Then use Exercise 7 in Chapter 37.)
