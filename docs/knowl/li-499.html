Item.

If \(\CB = \{\vb_1, \vb_2, \ldots, \vb_n\}\) and \(\CC = \{\vc_1, \vc_2, \ldots, \vc_n\}\) are two bases for \(\R^n\text{,}\) then the change of basis matrix from \(\CB\) to \(\CC\) is the matrix

\begin{equation*} \underset{\CC \leftarrow \CB}{P} = [[\vb_1]_{\CC} \ [\vb_2]_{\CC} \ \cdots \ [\vb_n]_{\CC}] \end{equation*}

that satisfies

\begin{equation*} [\vx]_{\CC} = \underset{\CC \leftarrow \CB}{P}[\vx]_{\CB} \end{equation*}

for any vector \(\vx\) in \(\R^n\text{.}\)
