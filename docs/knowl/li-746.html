Item.

If \(a_1\text{,}\) \(a_2\text{,}\) \(\ldots\text{,}\) \(a_n\) are positive scalars, then

\begin{equation*} \langle [u_1 \ u_2 \ \cdots \ u_n]^{\tr}, [v_1 \ v_2 \ \cdots \ v_n]^{\tr} \rangle = a_1u_1v_1+a_2u_2v_2+ \cdots + a_nu_nv_n \end{equation*}

defines an inner product on \(\R^n\text{.}\)
