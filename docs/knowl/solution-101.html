Solution 16.8.b.i.1.

Find the vector \(\vv\) given the basis \(\CB\) of \(V\) and the coordinate vector \([\vv]_{\CB}\text{.}\)

Since \([\vv]_{\CB} = [2 \ 1 \ 3]^{\tr}\text{,}\) it follows that

\begin{equation*} \vv = 2[1\ 0 \ 1]^{\tr} + [1 \ 1 \ 1]^{\tr} + 3[0 \ 1 \ 1]^{\tr} = [3 \ 4 \ 6]^{\tr}\text{.} \end{equation*}
