Since both \(\proj_W \vu\) and \(\vx\) are in \(W\text{,}\) we know that \(\proj_W \vu - \vx\) is in \(W\text{.}\) Since \(\proj_{W^{\perp}} \vu = \vu-\proj_W \vu\) is orthogonal to every vector in \(W\text{,}\) we have that \(\vu-\proj_W \vu\) is orthogonal to \(\proj_W \vu - \vx\text{.}\) We can now use the Generalized Pythagorean Theorem to conclude that

\begin{equation*} ||\vu - \vx||^2 = ||\vu-\proj_W \vu||^2 + ||\proj_W \vu - \vx||^2\text{.} \end{equation*}
