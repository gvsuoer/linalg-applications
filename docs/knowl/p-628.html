Note that to find the weights that make \(\vw\) a linear combination of the vectors \(\vv_1\) and \(\vv_2\text{,}\) we simply solved the linear system corresponding to the augmented matrix

\begin{equation*} [\vv_1 \ \vv_2 \ | \ \vw]\text{,} \end{equation*}

where the vectors \(\vv_1\text{,}\) \(\vv_2\text{,}\) and \(\vw\) form the columns of an augmented matrix, and the solution of the system gave us the weights of the linear combination. In general, if we want to find weights \(c_1\text{,}\) \(c_2\text{,}\) \(\ldots\text{,}\) \(c_m\) so that a vector \(\vw\) in \(\R^n\) is a linear combination of the vectors \(\vv_1\text{,}\) \(\vv_2\text{,}\) \(\ldots\text{,}\) \(\vv_m\) in \(\R^n\text{,}\) we solve the system corresponding to the augmented matrix

\begin{equation*} [\vv_1 \ \vv_2 \ \vv_3 \ \cdots \ \vv_m | \ \vw]\text{.} \end{equation*}
