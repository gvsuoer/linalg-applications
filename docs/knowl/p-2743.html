In general, as ActivityÂ 16.5 suggests, we can use the standard basis to do our work to find a change of basis matrix

\begin{equation*} \underset{\CC \leftarrow \CB}{P} = [[\vb_1]_{\CC} \ [\vb_2]_{\CC} \ \cdots \ [\vb_n]_{\CC}] \end{equation*}

from a basis \(\CB = \{\vb_1, \vb_2, \ldots, \vb_n\}\) of \(\R^n\) to a basis \(\CC = \{\vc_1, \vc_2, \ldots, \vc_n\}\) of \(\R^n\text{.}\) Recall that we need to write \(\vb_i\) as a linear combination of the vectors in \(\CC\text{.}\) That is, we need to find weights \(x_{1,i}\text{,}\) \(x_{2,i}\text{,}\) \(\ldots\text{,}\) \(x_{n,i}\) so that

\begin{equation} \vb_i = x_{1,i}\vc_1 + x_{2,i} \vc_2 + \cdots + x_{n,i} \vc_n\text{.}\tag{16.1} \end{equation}
