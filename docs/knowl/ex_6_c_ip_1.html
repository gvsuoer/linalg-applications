Exercise 4.

Define \(\langle \ , \ \rangle\) on \(\R^2\) by \(\langle [u_1 \ u_2]^\tr , [v_1 \ v_2]^\tr \rangle = 4u_1v_1+2u_2v_2\text{.}\)

(a)

Show that \(\langle \ , \ \rangle\) is an inner product.

(b)

The inner product \(\langle \ , \ \rangle\) can be represented as a matrix transformation \(\langle \vu, \vv \rangle = \vu^{\tr} A \vv\text{,}\) where \(\vu\) and \(\vv\) are written as column vectors. Find a matrix \(A\) that represents this inner product.
