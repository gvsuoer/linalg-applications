In ActivityÂ 23.2 we defined two vectors \(\vu\) and \(\vv\) in \(\R^n\) to be orthogonal (or perpendicular) if \(\vu \cdot \vv = 0\text{.}\) A use of orthogonality in geometry is to define a plane. A plane through the origin in \(\R^3\) is a two dimensional subspace of \(\R^3\text{,}\) and a plane is defined to be the set of all vectors in \(\R^3\) that are orthogonal to a given vector (called a normal vector). For example, to find the equation of the plane through the origin in \(\R^3\) orthogonal to the normal vector \(\vn = [1 \ 2 \ -1]^{\tr}\text{,}\) we seek all the vectors \(\vv = [x \ y \ z]^{\tr}\) such that

\begin{equation*} \vv \cdot \vn = \vzero\text{.} \end{equation*}
