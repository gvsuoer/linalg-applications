Define \(\{\psi_{0,0}\}\) as

\begin{equation*} \psi_{0,0}(x) = \begin{cases}1 \amp \text{ if } 0 \leq x \lt \frac{1}{2} \\ -1 \amp \text{ if } \frac{1}{2} \leq x \lt 1 \\ 0 \amp \text{ otherwise } \end{cases}\text{.} \end{equation*}

A picture of \(\psi_{0,0}\) is shown in Figure 32.13. Since \(\psi_{0,0}\) assumes values of \(1\) and \(-1\text{,}\) we can use \(\psi_{0,0}\) to perform differencing. The function \(\psi = \psi_{0,0}\) is the Mother Haar wavelet. 56  Show that \(\{\varphi, \psi\}\) is a basis for \(V_1\text{.}\)
