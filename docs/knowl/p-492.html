A linear system is inconsistent if and only if a row echelon form of the augmented matrix of the system contains a row of the form

\begin{equation*} [0 \ 0 \ 0 \ \cdots \ 0 \ *]\text{,} \end{equation*}

where \(*\) is not zero. Another way to say this is that a linear system is inconsistent if and only if the last column of the augmented matrix of the system is a pivot column.
