Task 35.18.a.

Show that the set \(\CB = \{p_1(t), p_2(t), p_3(t), p_4(t)\}\) is an orthogonal basis for \(V\text{.}\)

Solution.

All calculations are done by hand or with a computer algebra system, so we leave those details to the reader.

If we show that the set \(\CB\) is an orthogonal set, then TheoremÂ 35.8 shows that \(\CB\) is linearly independent. Since \(\dim(\pol_3) = 4\text{,}\) the linearly independent set \(\CB\) that contains four vectors must be a basis for \(\pol_3\text{.}\) To determine if the set \(\CB\) is an orthogonal set, we must calculate the inner products of pairs of distinct vectors in \(\CB\text{.}\) Since \(\langle 1+t,1-3t \rangle = 0\text{,}\) \(\langle 1+t,3t-5t^3 \rangle = 0\text{,}\) \(\langle 1+t,1-3t^2 \rangle = 0\text{,}\) \(\langle 1-3t,3t-5t^3 \rangle = 0\text{,}\) \(\langle 1-3t,1-3t^2 \rangle = 0\text{,}\) and \(\langle 3t-5t^3,1-3t^2 \rangle = 0\text{,}\) we conclude that \(\CB\) is an orthogonal basis for \(\pol_3\text{.}\)
