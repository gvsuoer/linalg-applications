Theorem 11.1. The Invertible Matrix Theorem.

Let \(A\) be an \(n \times n\) matrix. The following statements are equivalent:

  1. \(A\) is an invertible matrix.

  2. The equation \(A \vx = \vzero\) has only the trivial solution.

  3. \(A\) has \(n\) pivot columns.

  4. The columns of \(A\) span \(\R^n\text{.}\)

  5. \(A\) is row equivalent to the identity matrix \(I_n\text{.}\)

  6. The columns of \(A\) are linearly independent.

  7. The columns of \(A\) form a basis for \(\R^n\text{.}\)

  8. The matrix transformation \(T\) from \(\R^n\) to \(\R^n\) defined by \(T(\vx) = A\vx\) is one-to-one.

  9. The matrix equation \(A \vx = \vb\) has exactly one solution for each vector \(\vb\) in \(\R^n\text{.}\)

  10. The matrix transformation \(T\) from \(\R^n\) to \(\R^n\) defined by \(T(\vx) = A\vx\) is onto.

  11. There is an \(n \times n\) matrix \(C\) so that \(AC = I_n\text{.}\)

  12. There is an \(n \times n\) matrix \(D\) so that \(DA = I_n\text{.}\)

  13. The scalar 0 is not an eigenvalue of \(A\text{.}\)

  14. \(A^{\tr}\) is invertible.
