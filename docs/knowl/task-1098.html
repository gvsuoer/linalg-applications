Task 14.b.

The first part of this exercise presents an example of a matrix that satisfies its own characteristic equation. Show that if \(A\) is an \(n \times n\) diagonalizable matrix with characteristic polynomial \(p(x)\text{,}\) then \(p(A) = 0\text{.}\) 34  That is, if \(p(x) = a_nx^n+a_{n-1}x^{n-1} + \cdots + a_1x + a_0\text{,}\) then \(p(A) = a_nA^n+a_{n-1}A^{n-1} + \cdots + a_1A + a_0 = 0\text{.}\) (Hint: If \(A = PDP^{-1}\) for some diagonal matrix \(D\text{,}\) show that \(p(A) = Pp(D)P^{-1}\text{.}\) Then determine \(p(D)\text{.}\))
