Definition 1.5.

A solution to a system of linear equations is an ordered \(n\)-tuple \((s_1, s_2, \ldots, s_n)\) of numbers so that we obtain all true statements in the system when we replace the variable in order with \(s_1\text{,}\) \(s_2\text{,}\) \(\ldots\text{,}\) and \(s_n\text{.}\)
