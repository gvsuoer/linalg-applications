We need to show that \(T\) is a linear transformation, and that \(T\) is both one-to-one and onto. We start by demonstrating that \(T\) is a linear transformation. Let \(A = [a_{ij}]\) and \(B = [b_{ij}]\) be in \(\M_{2 \times 2}\text{.}\) Then

\begin{align*} T(A+B) \amp = T\left([a_{ij}+b_{ij}]\right)\\ \amp = \left(a_{11}+b_{11}\right) + \left(a_{12}+b_{12}\right)t + \left(a_{21}+b_{21}\right)t^2 + \left(a_{22}+b_{22}\right)t^3\\ \amp = \left(a_{11}+a_{12}t+a_{21}t^2+a_{22}t^3\right) + \left(b_{11}+b_{12}t+b_{21}t^2+b_{22}t^3\right)\\ \amp = T(A) + T(B)\text{.} \end{align*}
