Project Activity 4.7.

Given any position \((a,b)\text{,}\) we want to know if our knight can move from its start position \((1,0)\) to position \((a,b)\text{.}\)

(a)

Write a vector equation whose solution will tell us if it is possible for our knight to move from its start position \((1,0)\) to position \((a,b)\text{.}\)

(b)

Show that the solution to part (a) can be written in the form

\begin{align} x_1 \amp = \frac{1}{4}\left(-5x_3+3x_4+b+2(a-1)\right)\tag{4.3}\\ x_2 \amp = \frac{1}{4}\left(3x_3-5x_4+b-2(a-1)\right)\tag{4.4}\\ x_3 \amp \text{ is free }\notag\\ x_4 \amp \text{ is free. }\notag \end{align}
