To find weights \(c_1\text{,}\) \(c_2\text{,}\) \(\ldots\text{,}\) \(c_m\) so that a vector \(\vw\) in \(\R^n\) is a linear combination of the vectors \(\vv_1\text{,}\) \(\vv_2\text{,}\) \(\ldots\text{,}\) \(\vv_m\) in \(\R^n\text{,}\) we simply solve the system corresponding to the augmented matrix

\begin{equation*} [\vv_1 \ \vv_2 \ \vv_3 \ \cdots \ \vv_m | \ \vw]\text{.} \end{equation*}
