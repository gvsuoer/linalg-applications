If \(g\) is an arbitrary function in \(C[-\pi,\pi]\text{,}\) we will write the projection of \(g\) onto \(W_n\) as

\begin{align*} a_0\left(\frac{1}{\sqrt{2}}\right) + a_1 \cos(x) + b_1\sin(x) + a_2 \cos(2x) \amp + b_2 \sin(2x) + \cdots\\ \amp + a_n \cos(nx) + b_n \sin(nx)\text{.} \end{align*}
