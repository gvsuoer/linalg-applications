We learned in Preview ActivityÂ 8.1 that we can add two matrices of the same size together by adding corresponding entries and we can multiply any matrix by a scalar by multiplying each entry of the matrix by that scalar. More generally, if \(A = [a_{ij}]\) and \(B = [b_{ij}]\) are \(m \times n\) matrices and \(c\) is any scalar, then

\begin{equation*} A + B = [a_{ij}+b_{ij}] \ \ \text{ and } \ \ cA = [ca_{ij}]\text{.} \end{equation*}
