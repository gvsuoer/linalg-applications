Exercise 8.

The trace of an \(n \times n\) matrix \(A = [a_{ij}]\) has some useful properties.

(a)

Show that \(\trace(A+B) = \trace(A) + \trace(B)\) for any \(n \times n\) matrices \(A\) and \(B\text{.}\)

(b)

Show that \(\trace(cA) = c \trace(A)\) for any \(n \times n\) matrix \(A\) and any scalar \(c\text{.}\)

(c)

Show that \(\trace\left(A^{\tr}\right) = \trace(A)\) for any \(n \times n\) matrix.
