Activity 34.3.

Let \(V\) be a vector space with an ordered basis \(\CB = \{\vv_1, \vv_2, \ldots, \vv_n\}\text{.}\) Then \(T\) maps \(V\) into \(\R^n\text{.}\) We want to show that \(T\) maps \(V\) onto \(\R^n\text{.}\) Recall that a function \(f\) from a set \(X\) to a set \(Y\) is onto if for any element \(y\) in \(Y\text{,}\) there is an element \(x\) in \(X\) such that \(f(x) = y\text{.}\) Let \(\vb = [b_1 \ b_2 \ \ldots \ b_n]^{\tr}\) be a vector in \(\R^n\text{.}\) Must there be a vector \(\vv\) in \(V\) so that \(T(\vv) = \vb\text{?}\) If so, find such a vector. If not, explain why not.
