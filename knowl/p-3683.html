An elementary matrix \(E\) obtained by multiplying a row by \(r\) is a diagonal matrix with one \(r\) along the diagonal and the rest 1s, so \(\det(E) = r\text{.}\) Similarly, an elementary matrix \(E\) obtained by adding a multiple of a row to another is a triangular matrix with 1s along the diagonal, so \(\det(E) = 1\text{.}\) The fact that the the determinant of an elementary matrix obtained by swapping two rows is \(-1\) is a bit more complicated and is verified independently later in this section.
