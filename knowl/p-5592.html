To obtain the formal definition of a basis, which is a minimal spanning set, we consider what additional property makes a spanning set a minimal spanning set. As a consequence of TheoremÂ 32.5, if \(S\) is a spanning set that is linearly dependent, then we can find a proper subset of \(S\) that has the same span. Thus, the set \(S\) cannot be a minimal spanning set. However, if \(S\) is linearly independent, then no vector in \(S\) is a linear combination of the others and we need all of the vectors in \(S\) to form the span.
