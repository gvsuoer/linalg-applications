Task 10.1.b.iii.

A matrix inverse is extremely useful in solving matrix equations and can help us in solving systems of equations. Suppose that \(A\) is an invertible matrix, i.e., there exists \(A^{-1}\) such that \(AA^{-1}=A^{-1}A=I_n\text{.}\)

(A)

Consider the system \(A\vx=\vb\text{.}\) Use the inverse of \(A\) to show that this system has a solution for every \(\vb\) and find an expression for this solution in terms of \(\vb\) and \(A^{-1}\text{.}\) (Note that since matrix multiplication is not commutative, we have to pay attention to the order in which we multiply matrices. For example, \(A^{-1}AB=B\) while we cannot simplify \(ABA^{-1}\) to \(B\) unless \(A\) and \(B\) commute.)

(B)

If \(A\text{,}\) \(B\text{,}\) and \(C\) are matrices and \(A+C = B+C\text{,}\) then we can subtract the matrix \(C\) from both sides to see that \(A = B\text{.}\) We saw in SectionÂ 8 that there is no corresponding general cancellation property for matrix multiplication when we found that \(AB=AC\) could hold while \(B\neq C\text{.}\) However, we can cancel \(A\) from this equation in certain circumstances. Suppose that \(AB=AC\) and that \(A\) is an invertible matrix. Show that we can cancel \(A\) in this case and conclude that \(B=C\text{.}\) (Note: When simplifying the product of matrices, again keep in mind that matrix multiplication is not commutative.)
