In Preview ActivityÂ 4.1 we considered scalar multiples of a vector in \(\R^2\text{.}\) The arrow representation helps in visualizing scalar multiples as well. Geometrically, a scalar multiple \(c\vv\) of a nonzero vector \(\vv\) is a vector in the same direction as \(\vv\) if \(c>0\) and in the opposite direction as \(\vv\) if \(c\lt 0\text{.}\) If \(c>1\text{,}\) scalar multiplication stretches the vector, while \(0\lt c\lt 1\) shrinks the vector. We also saw that the collection of all scalar multiples of a vector \(\vv\) in \(\R^2\) gives us a line through the origin and \(\vv\text{,}\) except when \(\vv=\vzero\) in which case we only obtain \(\vzero\text{.}\) In other words, for a nonzero vector \(\vv\text{,}\) the set \(S = \{c\vv : c \text{ is a scalar } \}\) is the line through the origin and \(\vv\) in \(\R^2\text{.}\)
