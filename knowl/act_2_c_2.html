Activity 10.3.

Suppose that \(A\) is an invertible \(n \times n\) matrix. Hence we have an inverse matrix \(A^{-1}\) for which \(AA^{-1}=A^{-1}A=I_n\text{.}\) We will see how the inverse is useful in solving matrix equations involving \(A\text{.}\)

(a)

Explain why the matrix expressions

\begin{equation*} \ A^{-1}(AB) , \ A^{-1}(A(BA)A^{-1}) \ \text{ and } \ BA^{-1}BAA^{-1}B^{-1}A \end{equation*}

can all be simplified to \(B\text{.}\)

Hint.
(b)

Suppose the system \(A \vx = \vb\) has a solution. Explain why then \(A^{-1}(A\vx)= A^{-1}\vb\text{.}\) What does this equation simplify to?

(c)

Since we found one single expression for the solution \(\vx\) in equation \(A\vx=\vb\text{,}\) this implies that the equation has a unique solution. What does this imply about the matrix \(A\text{?}\)
