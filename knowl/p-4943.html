Preview ActivityÂ 29.3 contains the basic ideas behind the Singular Value Decomposition. Let \(A\) be an \(m \times n\) matrix with real entries. Note that \(A^{\tr}A\) is a symmetric \(n \times n\) matrix and, hence, it can be orthogonally diagonalized. Let \(V = [\vv_1 \ \vv_2 \ \vv_3 \ \cdots \ \vv_n ]\) be an \(n \times n\) orthogonal matrix whose columns form an orthonormal set of eigenvectors for \(A^{\tr}A\text{.}\) For each \(i\text{,}\) let \((A^{\tr}A)\vv_i = \lambda_i \vv_i\text{.}\) We know

\begin{equation*} V^{\tr} (A^{\tr}A) V = \left[ \begin{array}{ccccc} \lambda_1\amp 0\amp 0\amp \cdots\amp 0 \\ 0\amp \lambda_2\amp 0\amp \cdots\amp 0 \\ \vdots \amp \vdots \amp \vdots \amp \amp \vdots \\ 0\amp 0\amp 0\amp \cdots\amp \lambda_n \end{array} \right]\text{.} \end{equation*}
