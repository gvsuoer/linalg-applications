Solution.

Using technology, we find that the reduced row echelon form of the augmented matrix for this system is

\begin{equation*} \left[ \begin{array}{cccc|r} 1\amp 0\amp 0\amp 0\amp 3 \\ 0\amp 1\amp 0\amp 0\amp 7 \\ 0\amp 0\amp 1\amp 0\amp -2 \\ 0\amp 0\amp 0\amp 1\amp 4 \end{array} \right]\text{.} \end{equation*}

So the solution to this system is \(x_1 = 3\text{,}\) \(x_2 = 7\text{,}\) \(x_3 = -2\text{,}\) and \(x_4 = 4\text{.}\)
