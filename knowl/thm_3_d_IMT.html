Theorem 15.6. The Invertible Matrix Theorem.

Let \(A\) be an \(n \times n\) matrix. The following statements are equivalent.

  1. The matrix \(A\) is an invertible matrix.

  2. The matrix equation \(A \vx = \vzero\) has only the trivial solution.

  3. The matrix \(A\) has \(n\) pivot columns.

  4. Every row of \(A\) contains a pivot.

  5. The columns of \(A\) span \(\R^n\text{.}\)

  6. The matrix \(A\) is row equivalent to the identity matrix \(I_n\text{.}\)

  7. The columns of \(A\) are linearly independent.

  8. The columns of \(A\) form a basis for \(\R^n\text{.}\)

  9. The matrix transformation \(T\) from \(\R^n\) to \(\R^n\) defined by \(T(\vx) = A\vx\) is one-to-one.

  10. The matrix equation \(A \vx = \vb\) has exactly one solution for each vector \(\vb\) in \(\R^n\text{.}\)

  11. The matrix transformation \(T\) from \(\R^n\) to \(\R^n\) defined by \(T(\vx) = A\vx\) is onto.

  12. There is an \(n \times n\) matrix \(C\) so that \(AC = I_n\text{.}\)

  13. There is an \(n \times n\) matrix \(D\) so that \(DA = I_n\text{.}\)

  14. The scalar 0 is not an eigenvalue of \(A\text{.}\)

  15. The matrix \(A^{\tr}\) is invertible.

  16. \(\Nul A = \{\vzero\}\text{.}\)

  17. \(\Col A = \R^n\text{.}\)

  18. \(\displaystyle \dim(\Col A) = n\)

  19. \(\displaystyle \dim(\Nul A) = 0\)

  20. \(\displaystyle \rank(A) = n\)
