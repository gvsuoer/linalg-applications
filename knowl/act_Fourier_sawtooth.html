Project Activity 35.15.

Let \(f\) be defined by \(f(x) = x\) on \([-\pi,\pi]\) and repeated periodically afterwards with period \(2\pi\text{.}\) Let \(p_n\) be the projection of \(f\) onto \(W_n\text{.}\)

(a)

Evaluate the integrals to find the projection \(p_1\text{.}\)

(b)

Use appropriate technology to find the projections \(p_{10}\text{,}\) \(p_{20}\text{,}\) and \(p_{30}\) for the sawtooth function \(f\text{.}\) Draw pictures of these approximations against \(f\) and explain what you see.

(c)

Now we find formulas for all the Fourier coefficients. Use the fact that \(x \cos(mx)\) is an odd function to explain why \(a_m = 0\) for each \(m\text{.}\) Then show that \(b_m = (-1)^{m+1}\frac{2}{m}\) for each \(m\text{.}\)

(d)

Go back to the website falstad.com/fourier/ and replay the sawtooth tone. Explain what the white buttons represent.
