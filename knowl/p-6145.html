We have seen, e.g., linear regression to fit a line to a set of data, that we often want to find a vector in a subspace that “best” approximates a given vector in a vector space. As in \(\R^n\text{,}\) the projection of a vector onto a subspace has this important property. That is, \(\proj_W \vv\) is the vector in \(W\) closest to \(\vv\) and therefore the best approximation of \(\vv\) by a vector in \(W\text{.}\) To see that this is true in any inner product space, we first need a generalization of the Pythagorean Theorem that holds in inner product spaces.
