Example 35.18.

Let \(V = \pol_3\) be the inner product space with inner product

\begin{equation*} \langle p(t), q(t) \rangle = \int_{-1}^1 p(t)q(t) \ dt\text{.} \end{equation*}

Let \(p_1(t) = 1+t\text{,}\) \(p_2(t) = 1-3t\text{,}\) \(p_3(t) = 3t-5t^3\text{,}\) and \(p_4(t) = 1-3t^2\text{.}\)

(a)

Show that the set \(\CB = \{p_1(t), p_2(t), p_3(t), p_4(t)\}\) is an orthogonal basis for \(V\text{.}\)

Solution.

All calculations are done by hand or with a computer algebra system, so we leave those details to the reader.

If we show that the set \(\CB\) is an orthogonal set, then Theorem 35.8 shows that \(\CB\) is linearly independent. Since \(\dim(\pol_3) = 4\text{,}\) the linearly independent set \(\CB\) that contains four vectors must be a basis for \(\pol_3\text{.}\) To determine if the set \(\CB\) is an orthogonal set, we must calculate the inner products of pairs of distinct vectors in \(\CB\text{.}\) Since \(\langle 1+t,1-3t \rangle = 0\text{,}\) \(\langle 1+t,3t-5t^3 \rangle = 0\text{,}\) \(\langle 1+t,1-3t^2 \rangle = 0\text{,}\) \(\langle 1-3t,3t-5t^3 \rangle = 0\text{,}\) \(\langle 1-3t,1-3t^2 \rangle = 0\text{,}\) and \(\langle 3t-5t^3,1-3t^2 \rangle = 0\text{,}\) we conclude that \(\CB\) is an orthogonal basis for \(\pol_3\text{.}\)

(b)

Use 35.10 to write the polynomial \(q(t) = t^2+t^3\) as a linear combination of the basis vectors in \(\CB\text{.}\)

Solution.

All calculations are done by hand or with a computer algebra system, so we leave those details to the reader.

We can write the polynomial \(q(t) = 1+t+t^2+t^3\) as a linear combination of the basis vectors in \(\CB\) as follows:

\begin{align*} q(t) = \frac{\langle q(t),p_1(t) \rangle }{\langle p_1(t),p_1(t) \rangle} p_1(t) \amp + \frac{\langle q(t),p_2(t) \rangle }{\langle p_2(t),p_2(t) \rangle} p_2(t)\\ \amp + \frac{\langle q(t),p_3(t) \rangle }{\langle p_3(t),p_3(t) \rangle} p_3(t) + \frac{\langle q(t),p_4(t) \rangle }{\langle p_4(t),p_4(t) \rangle} p_4(t)\text{.} \end{align*}

Now

\begin{equation*} \begin{array}{ccc} \langle q(t), p_1(t) \rangle = \frac{16}{15}, \amp \langle q(t), p_2(t) \rangle = -\frac{8}{15}, \amp \langle q(t), p_3(t) \rangle = -\frac{8}{35}, \\ \langle q(t), p_4(t) \rangle = -\frac{8}{15}, \amp \langle p_1(t), p_1(t)\rangle = \frac{8}{3}, \amp \langle p_2(t), p_2(t) \rangle = 8, \\ \langle p_3(t), p_3(t) \rangle = \frac{8}{7}, \amp \langle p_4(t), p_4(t) \rangle = \frac{8}{5} \amp \end{array} \end{equation*}

so

\begin{align*} q(t) \amp = \frac{ \frac{16}{15} }{ \frac{8}{3} } p_1(t) - \frac{ \frac{8}{15} }{ 8 } p_2(t) - \frac{ \frac{8}{35} }{ \frac{8}{7} } p_3(t) - \frac{ \frac{8}{15} }{ \frac{8}{5} } p_4(t)\\ \amp \frac{2}{5} p_1(t) - \frac{1}{15}p_2(t) - \frac{1}{5} p_3(t) - \frac{1}{3} p_4(t)\text{.} \end{align*}
