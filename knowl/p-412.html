Since there are leading 1s in the first three columns, we can use those entries to write \(x_1\text{,}\) \(x_2\text{,}\) and \(x_3\) in terms of \(x_4\text{.}\) We then choose \(x_4\) to be arbitrary and write the remaining variables in terms of \(x_4\text{.}\) Let \(x_4 = t\text{.}\) Solving the third equation for \(x_3\) gives us \(x_3 = 4+2t\text{.}\) The second equation shows that \(x_2 = 2+t\text{,}\) and the first that \(x_1 = 0\text{.}\) Each value of \(t\) provides a solution to the system, so our system has infinitely many solutions. These solutions are

\begin{equation*} x_1 = 0, \ x_2 = 2+t, \ x_3 = 4+2t, \ \text{ and } \ x_4 = t\text{,} \end{equation*}

where \(t\) can have any value.
