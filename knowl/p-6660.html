A matrix transformation \(T\) from \(\R^n\) to \(\R^m\) is a linear transformation defined by \(T(\vx) = A\vx\) for some \(m \times n\) matrix \(A\text{.}\) We can use the matrix to quickly determine if \(T\) is one-to-one or onto â€” if every column of \(A\) contains a pivot, then \(T\) is one-to-one, and if every row of \(A\) contains a pivot, then \(T\) is onto. As we will see, we can represent any linear transformation between finite dimensional vector spaces as a matrix transformation, which will allow us to use all of the tools we have developed for matrices to study linear transformations. We will begin with linear transformations from \(\R^n\) to \(\R^m\text{.}\)
