In \(\R^n\) using the dot product as inner product, if \(\vv = [v_1 \ v_2 \ v_3 \ \ldots \ v_n]^{\tr}\) and \(\proj_W \vv = [w_1 \ w_2 \ w_3 \ \ldots \ w_n]^{\tr}\text{,}\) then the square of the error in approximating \(\vv\) by \(\proj_W \vv\) is given by

\begin{equation*} || \vv - \proj_W \vv ||^2 = \sum_{i=1}^n (v_i - w_i)^2\text{.} \end{equation*}
