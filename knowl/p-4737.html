Preview ActivityÂ 28.1 contains the main tool that we need to convert a quadratic form into one of these standard forms. By this we mean that if we have a quadratic form \(Q\) in the variables \(x_1\text{,}\) \(x_2\text{,}\) \(\ldots\text{,}\) \(x_n\text{,}\) we want to find variables \(y_1\text{,}\) \(y_2\text{,}\) \(\ldots\text{,}\) \(y_n\) in terms of \(x_1\text{,}\) \(x_2\text{,}\) \(\ldots\text{,}\) \(x_n\) so that when written in terms of the variables \(y_1\text{,}\) \(y_2\text{,}\) \(\ldots\text{,}\) \(y_n\) the quadratic form \(Q\) contains no cross terms. In other words, we want to find a vector \(\vy = \left[ \begin{array}{c} y_1 \\ y_2 \\ \vdots \\ y_n \end{array} \right]\) so that \(Q(\vx) = \vy^{\tr} D \vy\text{,}\) where \(D\) is a diagonal matrix.
