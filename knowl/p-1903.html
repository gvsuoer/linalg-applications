The Invertible Matrix Theorem is a theorem that provides many different statements that are equivalent to a matrix being invertible. As discussed in Preview ActivityÂ 11.1, two statements are said to be equivalent if, whenever one of the statements is true, then the other is also true. So to demonstrate, say, statements I and II are equivalent, we need to argue that

  • if statement I is true, then so is statement II, and

  • if statement II is true then so is statement I.
