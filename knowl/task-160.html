Task 3.9.b.

To burn glucose, we need to add oxygen to make the combustion happen. Carbon dioxide is different in that it can burn without the presence of oxygen. For example, when we mix magnesium (Mg) with dry ice (\(\text{ CO } _2\)), the products are magnesium oxide (MgO) and carbon (C). This is an interesting reaction to watch: you can see it at many websites, e.g., ebaumsworld.com/video/watch/404311/ or youtube.com/watch?v=-6dfi8LyRLA Use the method determined above to balance the chemical reaction

\begin{equation} \text{ Mg } + \text{ CO } _2 \to \text{ MgO } + \text{C}\text{.}\tag{3.4} \end{equation}
