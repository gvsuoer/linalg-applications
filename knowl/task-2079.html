Task 35.19.b.

Express the vector \([2 \ 0 \ -1 \ 3]^{\tr}\) as the sum of a vector in \(W\) and a vector orthogonal to \(W\text{.}\)

If \(\vv = [2 \ 0 \ -1 \ 3]^{\tr}\text{,}\) then \(\proj_{W} \vv\) is in \(W\) and

\begin{equation*} \proj_{W^{\perp}} \vv = \vv - \proj_{W} \vv = \left[-\frac{10}{3} \ \frac{13}{9} \ \frac{10}{9} \ -\frac{14}{9} \right] \end{equation*}

is in \(W^{\perp}\text{,}\) and \(\vv = \proj_{W} \vv + \proj_{W^{\perp}} \vv\text{.}\)
