Some quadratic forms form equations in \(\R^2\) that are very familiar: \(x^2+y^2=1\) is an equation of a circle, \(2x^2+3y^2=2\) is an equation of an ellipse, and \(x^2-y^2=1\) is an equation of a hyperbola. Of course, these do not represent all of the quadratic forms in \(\R^2\) â€” some contain cross-product terms. We can recognize the equations above because they contain no cross-product terms (terms involving \(xy\)). We can more easily recognize the quadratic forms that contain cross-product terms if we can somehow rewrite the forms in a different format with no cross-product terms. We illustrate how this can be done with the quadratic form \(Q\) defined by \(Q(\vx) = x^2-xy+y^2\text{.}\)
