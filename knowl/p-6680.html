If \(V=W\) and \(\CC = \CB\text{,}\) then we use the notation \([T]_{\CB}\) as a shorthand for \([T]_{\CB}^{\CC}\text{.}\) The matrix \([T]_{\CB}^{\CC}\) has the property that

\begin{equation*} [T(\vv)]_{\CC} = A [\vv]_{\CB} \end{equation*}

for any \(\vv\) in \(V\text{.}\) Recall that we can find the unique vector in \(W\) whose coordinate vector is \([T(\vv)]_{\CC}\text{,}\) so we have completely realized the transformation \(T\) as a matrix transformation. In essence, we are viewing \(T\) as a composite of linear transformations, first from \(V\) to \(\R^n\text{,}\) then from \(\R^n\) to \(\R^m\text{,}\) then from \(\R^m\) to \(W\) as illustrated in FigureÂ 38.3.
