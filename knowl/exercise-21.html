Exercise 4.

Consider the following system:

\begin{alignat*}{5} {}x \amp {}-{} \amp {2}y \amp {}+{} \amp {}z \amp {}={} \amp \ {-}\amp 1\amp {}\\ {} \amp {} \amp {2}y \amp {}-{} \amp {4}z \amp {}={} \amp \ {}\amp 6\amp {}\\ {} \amp {} \amp {h}y \amp {}-{} \amp {2}z \amp {}={} \amp \ {}\amp 1\amp {.} \end{alignat*}
(a)

Find a row echelon form of the augmented matrix for this system.

(b)

For which values of \(h\text{,}\) if any, does the system have (i.) no solutions, (ii.) exactly one solution, (iii.) infinitely many solutions? Find the solutions in each case.
