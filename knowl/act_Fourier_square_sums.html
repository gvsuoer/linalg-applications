Project Activity 35.16.

This activity is not connected to the idea of musical tones, so can be safely ignored if so desired. We conclude with a derivation of a very fascinating formula that you may have seen for \(\ds \sum_{n=1}^{\infty} \frac{1}{n^2}\text{.}\) To do so, we need to analyze the error in approximating a function \(g\) with a function in \(W_n\text{.}\)

Let \(p_n\) be the projection of \(g\) onto \(W_n\text{.}\) Notice that \(p_n\) is also in \(W_{n+1}\text{.}\) It is beyond the scope of this project, but in “nice” situations we have \(||g - p_n|| \to 0\) as \(n \to \infty\text{.}\) Now \(g - p_n\) is orthogonal to \(p_n\text{,}\) so the Pythagorean theorem shows that

\begin{equation*} ||g-p_n||^2 + ||p_n||^2 = ||g||^2\text{.} \end{equation*}

Since \(||g-p_n||^2 \to 0\) as \(n \to \infty\text{,}\) we can conclude that

\begin{equation} \lim_{n \to \infty} ||p_n||^2 = ||g||^2\text{.}\tag{35.11} \end{equation}

We use these ideas to derive a formula for \(\ds \sum_{n=1}^{\infty} \frac{1}{n^2}\text{.}\)

(a)

Use the fact that \(\CB_n\) is an orthonormal basis to show that

\begin{equation*} ||p_n||^2 = a_0^2 +a_1^2 + b_1^2 + \cdots + a_n^2 + b_n^2\text{.} \end{equation*}

Conclude that

\begin{equation} ||g||^2 = a_0^2 +a_1^2 + b_1^2 + \cdots + a_n^2 + b_n^2 + \cdots\text{.}\tag{35.12} \end{equation}
(b)

For the remainder of this activity, let \(f\) be the sawtooth function defined by \(f(x) = x\) on \([-\pi,\pi]\) and repeated periodically afterwards. We determined the Fourier coefficients \(a_i\) and \(b_j\) of this function in Project Activity 35.15.

(i)

Show that

\begin{equation*} a_0^2 +a_1^2 + b_1^2 + \cdots + a_n^2 + b_n^2 + \cdots = 4\sum_{n=1}^{\infty} \frac{1}{n^2}\text{.} \end{equation*}
(ii)

Calculate \(||f||^2\) using the inner product and compare to (35.12) to find a surprising formula for \(\sum_{n=1}^{\infty} \frac{1}{n^2}\text{.}\)
