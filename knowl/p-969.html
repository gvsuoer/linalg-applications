Using the third chemical solution (represented by \(\vv_3\)) uses more information than we actually need to make the desired 43% water, 48% benzene and 9% acid chemical solution because the vector \(\vv_3\) is redundant — all of the material we need to make \(\vv_3\) is contained in \(\vv_1\) and \(\vv_2\text{.}\) This is the basic idea behind linear independence — representing information in the most efficient way.
