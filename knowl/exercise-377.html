Exercise 13.

Let \(\R^3\) be the inner product space with inner product

\begin{equation*} \langle [u_1 \ u_2 \ u_3]^{\tr}, [v_1 \ v_2 \ v_3]^{\tr}\rangle = u_1v_1 + 2u_2v_2 + u_3v_3\text{.} \end{equation*}

Let \(\CB = \{[1 \ 1 \ 1]^{\tr}, [1 \ -1 \ 1]^{\tr}\}\) and let \(W = \Span \ \CB\) in \(\R^3\text{.}\)

(a)

Show that \(\CB\) is an orthogonal basis for \(W\text{,}\) using the given inner product.

(b)

Explain why the vector \(\vv = [0 \ 2 \ 2]^{\tr}\) is not in \(W\text{.}\)

Hint.
(c)

Find the vector in \(W\) that is closest to \(\vv\text{.}\) How close is this vector to \(\vv\text{?}\)
