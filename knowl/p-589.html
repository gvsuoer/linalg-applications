In mathematics, any time we define operations on objects, such as addition of vectors, we ask which properties the operation has. For example, one might wonder if \(\vu+\vv=\vv+\vu\) for any two vectors \(\vu, \vv\) of the same size. If this property holds, we say that the addition of vectors is a commutative operation. However, to verify this property we cannot use examples since the property must hold for any two vectors. For simplicity, we focus on two-dimensional vectors \(\vu = \left[ \begin{array}{c} u_1\\ u_2 \end{array} \right]\) and \(\vv = \left[ \begin{array}{c} v_1\\ v_2 \end{array} \right]\text{.}\) Using these arbitrary vectors, can we say that \(\vu+\vv=\vv+\vu\text{?}\) If so, justify. If not, give a counterexample. (Note: Giving a counterexample is the best way to justify why a general statement is not true.)
