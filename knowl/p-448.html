The fact that \(x_3\) is free means that we can choose any value for \(x_3\) that we like and obtain a specific solution to the system. For example, if \(x_3=-1\text{,}\) then we have the solution \(x_1 = 6\text{,}\) \(x_2 = 2\text{,}\) \(x_3 = -1\text{,}\) and \(x_4 = 0\text{.}\) Check this to be sure.
